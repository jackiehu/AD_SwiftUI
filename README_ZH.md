## 用法


## 安装

### cocoapods

1.在 Podfile 中添加 `pod 'AD_SwiftUI', :git => 'https://github.com/jackiehu/AD_SwiftUI.git'`

2.执行 `pod install 或 pod update`

3.导入 `import AD_SwiftUI`


### 手动集成

AD_SwiftUI 也支持手动集成，只需把Sources文件夹中的AD_SwiftUI文件夹拖进需要集成的项目即可
